## Basic embedding

To simply embed a PDF document, pass it to an `url` option.
This works only when the Storybook's webpack rule for PDF files remained.
No guarantees if you changed or removed it.

[Example](/?path=/story/docs-pdf-examples--embed-pdf)

```js
// myStories.stories.jsx
import myPdf from '/path/to/file.pdf'

export const myStory = () => <MyComponent />

myStory.story = {
  parameters: {
    design: {
      type: 'pdf',
      url: myPdf,
    },
  },
}
```

## Changing initial page to show

You can specify which page to display at first.

[Example](/?path=/story/docs-pdf-examples--set-page)

```js
// myStories.stories.jsx
import myPdf from '/path/to/file.pdf'

export const myStory = () => <MyComponent />

myStory.story = {
  parameters: {
    design: {
      type: 'pdf',
      url: myPdf,
      page: 2,
    },
  },
}
```

## Setting scale and offset

You can tweak the default preview viewport by providing scale and offset values.

[Example](/?path=/story/docs-pdf-examples--set-scale-and-offset)

```js
// myStories.stories.jsx
import myPdf from '/path/to/file.pdf'

export const myStory = () => <MyComponent />

myStory.story = {
  parameters: {
    design: {
      type: 'pdf',
      url: myPdf,
      offset: [-300, 140],
      scale: 2,
    },
  },
}
```
