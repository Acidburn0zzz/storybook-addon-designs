# Embedding PDF documents

<Subtitle>
  This document describes how to embed PDF documents to your Storybook.
</Subtitle>

## Requirements

The addon uses `react-pdf` and will move it into optional dependencies in the future major versions.
If you want smooth transition in there, install the library manually.

## Usage

Set `type: "pdf"` and pass the PDF document.

```js
// myStories.stories.jsx
import myPdf from '/path/to/file.pdf'

export const myStory = () => <MyComponent />

myStory.story = {
  parameters: {
    design: {
      type: 'pdf',
      url: myPdf
    }
  }
}
```

You can find more detailed instructions [here](/?path=/docs/docs-pdf-examples--embed-pdf).

## Parameter options

| Option name                                      | Type               | Default value | Description                             |
| ------------------------------------------------ | ------------------ | ------------- | --------------------------------------- |
| `type` <Badge status="positive">required</Badge> | `"pdf"`            |               | Fixed parameter.                        |
| `url` <Badge status="positive">required</Badge>  | `string` (URL)     |               | An URL of the PDF document.             |
| `page`                                           | `number`           | `1`           | Initial page number.                    |
| `scale`                                          | `number`           | `1`           | Default scaling value for the document. |
| `offset`                                         | `[number, number]` | `[0, 0]`      | Position offsets (`[x, y]`).            |
